简记隐私权政策

生效日期：2026年5月7日

本隐私权政策适用于应用“简记”。简记是一款 Markdown 写作与预览工具。我们重视用户隐私，并尽量减少数据访问和数据收集。

1. 数据收集

简记不会主动收集、出售或共享用户的个人信息。用户创建、导入或编辑的 Markdown 内容主要保存在用户设备本地，开发者不会访问这些本地内容。

2. GitHub 功能

如果用户主动使用 GitHub 拉取或上传功能，应用会根据用户填写的 GitHub Token、仓库信息和文件路径，与 GitHub API 通信。

这些信息仅用于完成用户主动发起的 GitHub 文件读取和上传操作。GitHub Token 保存在用户设备本地，不会发送给开发者服务器。

GitHub 服务本身的数据处理规则请参考 GitHub 的隐私政策。

3. 网络访问

应用请求互联网权限，用于访问 GitHub API 以及完成用户主动触发的网络功能。

4. 第三方服务

简记不包含广告 SDK，不使用统计分析 SDK，也不使用开发者自建服务器收集用户数据。应用可能会在用户主动使用 GitHub 功能时连接 GitHub 服务。

5. 数据保存与删除

用户创建或导入的内容保存在本地设备中。用户可以通过清除应用数据、卸载应用，或在应用内删除相关内容来移除本地数据。

如果用户保存了 GitHub Token，可以在应用设置中清除，或通过清除应用数据删除。

6. 数据安全

简记不会将用户内容上传到开发者服务器。与 GitHub API 的通信通过 HTTPS 进行。

7. 儿童隐私

简记不是专门面向儿童的应用。我们不会有意收集儿童的个人信息。

9. 政策更新
